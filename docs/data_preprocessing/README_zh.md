# 数据集准备与预处理总览 (Overview)

[**English**](README_en.md) | **中文版**

本目录包含了本项目评估所用的各个下游任务的数据预处理与数据投毒流程。该流程旨在将各种不同格式的原始公开数据集统一转换为模型训练所需的标准 `.jsonl` 格式，并可选择性地为鲁棒性评估注入后门触发器。

## 通用工作流 (General Workflow)

1. **数据获取**: 将原始公开数据集（通常来源于 CodeXGLUE 仓库）下载到 `data/raw/` 目录中。
2. **标准预处理**: 运行预处理脚本，将原始数据进行提取、过滤，并转换成标准的 `train.jsonl`、`valid.jsonl` 和 `test.jsonl` 文件。
3. **数据投毒 (可选)**: 向处理好的数据集中注入后门触发器（例如使用隐蔽风格迁移 IST），生成投毒后的训练集和测试集。
4. **输出**: 最终可用于模型输入的数据集将被保存至 `data/processed/`（干净数据）或 `data/poisoned/`（含后门数据）。

---

## 支持的下游任务索引 (Task Index)

目前我们主要支持三个下游任务。每个任务都具有特定的数据格式、预处理细节和攻击策略。

请查阅各个任务的详细说明文档：

### 1. 缺陷检测 (Defect Detection, DD)
- **任务描述**: 一个二分类任务，旨在识别提供的代码片段是否存在漏洞/不安全 (1) 或安全 (0)。
- **数据集**: Devign
- **[查看 DD 详细指南](dd_preprocessing_zh.md)**

**数据集统计:**
| 数据集划分 | 样本数量 |
| :-------: | :------: |
| Train     | 21,854   |
| Valid     | 2,732    |
| Test      | 2,732    |

### 2. 代码克隆检测 (Clone Detection, CD)
- **任务描述**: 一个二分类任务，用于判断给定的两段代码是否为语义克隆（功能相同）。
- **数据集**: BigCloneBench
- **注意**: 为了计算效率，预处理包含了针对训练集和验证集的标准 10% 采样协议。
- **[查看 CD 详细指南](cd_preprocessing_zh.md)**

**数据集统计 (采样前的原始数据):**
| 数据集划分 | 样本数量 |
| :-------: | :------: |
| Train     | 901,028  |
| Valid     | 415,416  |
| Test      | 415,416  |

### 3. 代码修复/微调 (Code Refinement, CR)
- **任务描述**: 一个序列到序列 (seq2seq, 生成式) 任务，旨在自动修复 Java 代码中的 bug。模型需要学习从含有 bug 的代码到修复后代码的映射。
- **数据集**: CodeXGLUE Code Refinement (包含 Small 和 Medium 两个子集)
- **[查看 CR 详细指南](cr_preprocessing_zh.md)**

**数据集统计:**
| 数据集划分 | 样本数 (Small) | 样本数 (Medium) |
| :-------: | :------------: | :-------------: |
| Train     | 46,680         | 52,364          |
| Valid     | 5,835          | 6,545           |
| Test      | 5,835          | 6,545           |

### 4. XLCoST 指令防御数据 (XLCoST + CSA 12N)
- **任务描述**: 面向 C++ 指令防御训练的数据链路，包含 1N 抽取、12N 变体扩展、规则化清洗以及 FABE 评估接线。
- **数据集**: XLCoST C++ + HumanEval C++
- **[查看 XLCoST 详细指南](xlcost_preprocessing_zh.md)**

**默认规模:**
| 数据集划分 | 样本数量（默认） |
| :-------: | :-------------: |
| Train     | 1,000           |
| Eval      | 300             |
| Test      | 164 (HumanEval) |
